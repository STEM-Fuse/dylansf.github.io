# Style
## Different Styling Methods
### Inline
```css
<p style="font-family: sans-serif;">
```
### Internal
```css
<style>
p {
    font-family: sans-serif;
}
</style>
```
### External
```css
p {
    font-family: sans-serif;
}
```

## Feature List
* 3 Complete HTML pages

* use internal styling to style "Favorite Animal"
