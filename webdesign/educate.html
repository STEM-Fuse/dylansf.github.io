Welcome

Welcome to our updated Web Design Course! We are dedicated to providing a course that uses industry standard technology and delivers content knowledge applicable throughout the wide and changing world of web technologies.

The new course will cover HTML, CSS, and JavaScript and use these tools to build websites and explore programming. The course will focus on the development of real world portfolio projects that students can use to demonstrate their workplace skills.

Technologies Used

  • HTML -- major building block of the web since the beginning. We have updated this course to include a more comprehensive treatment of HTML.
  • CSS -- CSS styling remains as important as ever in designing the look and feel of websites.
  • JavaScript -- the scripting language of the web. allows us to explore object oriented programming with immediate feedback in a simple programming environment.

3rd Party Tools

This course will focus on development using modern web tools aligned to current industry standards.

  • Visual Studio Code — VSCode is a widely used, full featured editor that will give students a professional programming tool.
  • Google Chrome — modern web browser with built in developers tools.
  • Codepen.io — web development site that allows students to develop and view pages live.
