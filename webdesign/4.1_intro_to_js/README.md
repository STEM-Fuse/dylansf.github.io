# JavaScript

## Variables
* Declare Variables
```javascript
var myVariable;
```
* Assign Variables

```javascript
myVariable = 10;
```

* Data Types
```javascript
# Numeric Data type
myNumericVariable = 0.75;

# String Data type
myStringVariable = 'This is some text';

# boolean Data type
myBooleanVariable = true;
```
